# Explicit Feedback

Record feedback only when the user explicitly uses `PAI feedback`.

Supported forms:

```text
PAI feedback: pass
PAI feedback: pass - concise reason
PAI feedback: fail - what was missed
```

## Entry Format

```md
## YYYY-MM-DD - pass|fail

- Context: task, artifact, or skill
- Feedback: the user's concise reason, or `No reason provided`
- Follow-up: proposed change, or `None`
```
