# Start here — Claude Cowork

This page prevents Claude Cowork from confusing this template with another
product that also uses the initials PAI. It is only a Cowork starting guide.
Codex users can continue following the Codex instructions in `README.md`;
Codex behavior is controlled separately by `AGENTS.md`.

## Instructions for Claude

In this folder, **PAI** means only the Personal AI system contained in
`personal-ai-template-main`. It does not mean Daniel Miessler's Personal AI
Infrastructure, LifeOS, or any other outside system.

When the user asks for `PAI setup`:

1. Use only the files in this connected folder.
2. Do not search the web for a PAI system, download an installer, or install
   external software.
3. Read `CLAUDE.md`, then `ON.md`, then `skills/setup/SKILL.md`.
4. Begin the setup interview and ask one question at a time.

Do not ask the user which PAI system they meant. This folder is the complete
system they want to use.
